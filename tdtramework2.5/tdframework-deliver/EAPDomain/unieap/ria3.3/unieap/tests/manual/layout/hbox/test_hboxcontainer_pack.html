测试说明：
此测试用例用于验证HBoxContainer的pack属性。属性可选值为start 、center、end
  • start - 表示子容器从父容器内左侧开始定位,默认值
  • center - 表示子容器从父容器内中间开始定位
  • end - 表示子容器从父容器内右侧开始定位
子容器从父容器左侧开始定位,TitlePane的open属性设置为open,Grid高度为50%
子容器从父容器左侧开始定位,TitlePane的open属性设置为false
子容器从父容器中间开始定位,TitlePane的open属性设置为open，HBox嵌套树控件
子容器从父容器右侧开始定位,TitlePane的open属性设置为false，HBox嵌套树控件
功能点描述 操作过程 预期结果
支持嵌套的控件从父容器的左侧、中间、右侧定位 将pack属性分别设置为start、center、end 嵌套的子控件分别从父容器的左侧、中间、右侧定位
