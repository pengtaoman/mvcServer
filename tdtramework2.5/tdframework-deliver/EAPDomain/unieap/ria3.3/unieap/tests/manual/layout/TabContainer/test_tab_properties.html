该样例展示了Tab组件的如下特性：
1.支持onShow,onInit事件
2.支持herf,seleted,closable,refreshOnShow属性
3.支持cookie记忆Tab页选中状态
closable=true 可关闭
onShow事件，每次页面显示时触发
onInit事件，初始化时触发
onClose事件，关闭时时触发
默认选中第一页，当选中第二页后，session会记忆选中状态，刷新页面后，默认选中第二页（即刷新页面前的选中状态）

功能点描述 操作过程 预期结果
支持closable属性配置可关闭的Tab页 点击closable Tab页的'X' Tab页关闭
支持href属性的配置 点击 href Tab页 tab页内展示通过herf配置的URL的页面
支持refreshOnShow属性 1.点击 refreshOnShow Tab页
2.进行百度搜索
3.点击其他Tab页后再点击refreshOnShow Tab页		 Tab页显示百度主页，而不是搜索结果页
支持selected属性 无 进入页面后，Tab页被默认选中
支持onShow事件 点击onShowTab页 弹出alert('onShow');每次Tab页切换到页就被触发
支持onInit事件 点击onInitTab页 弹出alert('onInit');第一次点击到时触发，之后切换到页触发
支持onClose事件 点击oonClose Tab页的'x' 弹出alert('onClose');由于用例实现的onClose方法返回true，所以可以正常关闭，如果返回false则不会关闭
支持session记忆Tab页选中状态 点击 第二页 或 第三页 默认选中第一页，如果，选中第二页后，session会记忆选中状态，刷新页面后，默认选中第二页（即刷新页面前的选中状态）
